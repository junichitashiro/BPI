# MasterLens

エクセルのマスタファイルをブラウザで読み込み、列ごとにリアルタイムで絞り込むための、単一HTMLファイルのビューアです。「列が多く・行が多くて見づらいマスタ」から、必要な行だけをくっきり見せることを目的にしています。

サーバー不要・インストール不要。ファイルをダブルクリックでブラウザに開くだけで動きます。

## 特徴

- **ドラッグ&ドロップで読み込み** — `.xlsx` / `.xls` / `.csv` に対応
- **列ヘッダ個別フィルタ** — 各列の見出し下の欄に入力すると、その列だけで絞り込み（Excelのオートフィルタ感覚）
- **複数列のAND絞り込み** — 複数の列に入力すると、すべての条件を満たす行だけが残る
- **部分一致＋ハイライト** — 大文字小文字を区別せず「含む」で検索。一致箇所を強調表示
- **入力候補（オートコンプリート）** — 各列のマスタ内の値が候補として表示される
- **表示列の切り替え** — あまり参照しない列をワンクリックで隠せる
- **複数シート対応** — シートが2つ以上ある場合はシート選択が表示される
- **完全クライアントサイド** — データは外部に送信されず、ブラウザ内だけで処理される

## 使い方

1. `master_lens.html` をブラウザで開く
2. エクセルファイルをドロップする（または枠をクリックして選択）
3. 各列の見出し下の欄に絞り込みワードを入力する
4. 必要に応じて「表示する列」のチップで、見たくない列を隠す

複数の列に入力した場合は、入力したすべての列の条件を満たす行が表示されます。

## 動作要件

- モダンブラウザ（Chrome / Edge / Firefox / Safari の最新版）
- インターネット接続（SheetJSをCDNから読み込むため。社内マスタの内容自体は送信されません）

## 技術構成

| 項目 | 採用技術 |
|---|---|
| エクセル読み込み | [SheetJS (xlsx)](https://sheetjs.com/) ※CDN |
| 入力候補 | ネイティブの `<datalist>` |
| 絞り込み | 標準DOMイベント＋JavaScriptの `filter`（ライブラリなし） |
| 構成 | HTML / CSS / バニラJavaScript の単一ファイル |

外部依存はSheetJS（CDN）のみで、ビルド工程はありません。

## 仕様メモ

- **検索方式**：部分一致（含む）、大文字小文字は区別しない
- **複数列の条件**：AND（すべて満たす行のみ表示）
- **隠した列の扱い**：表示列から外した列は絞り込み条件として効かない。ただし入力値は保持され、再表示すると復活する
- **再描画**：入力中は本体（行）のみ更新し、ヘッダは作り直さないため入力欄のフォーカスが維持される
- **想定規模**：数百行程度のマスタを想定。全行を再描画する実装だが、この規模では体感上は瞬時
