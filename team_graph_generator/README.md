# チーム分析ツール

切削加工現場を想定したチーム分析ツールです。  
Excelファイルを入力として、人物相関図とスキルマップをHTMLファイルとして出力します。

## ファイル構成

```
.
├── team_graph.py   # 人物相関図ジェネレーター
├── skill_map.py    # スキルマップジェネレーター
├── team.xlsx       # 入力データ（サンプル）
├── team_graph.html # 出力：人物相関図（実行後に生成）
└── skill_map.html  # 出力：スキルマップ（実行後に生成）
```

## 必要環境

- Python 3.8 以上

## インストール

```bash
uv add pandas networkx pyvis openpyxl
```

## 使い方

```bash
# 人物相関図
python team_graph.py team.xlsx

# スキルマップ
python skill_map.py team.xlsx
```

出力された `.html` ファイルをブラウザで開いてください。  
引数を省略した場合はカレントディレクトリの `team.xlsx` を読み込みます。

## 入力データ（Excel）

`team.xlsx` に以下の3シートを用意してください。

### シート1：メンバー

| 列名 | 内容 | 例 |
|------|------|----|
| 氏名 | メンバーの名前 | 中川 |
| 役割 | 担当する役割 | 班長 |
| チーム | 所属チーム | A |

### シート2：スキル

| 列名 | 内容 | 例 |
|------|------|----|
| 氏名 | メンバーの名前 | 中川 |
| スキル | 保有するスキル名 | NC旋盤 |

1人が複数スキルを持つ場合は行を分けて記載してください。

### シート3：業務フロー

| 列名 | 内容 | 例 |
|------|------|----|
| 送り手 | 情報・指示を渡す側 | 小林 |
| 受け手 | 情報・指示を受ける側 | 中川 |
| 業務内容 | やり取りの内容 | 加工プログラム渡し |

## 出力内容

### 人物相関図（team_graph.html）

- 業務フローを有向グラフで可視化
- 媒介中心性スコアを計算し、ボトルネック候補を赤枠で強調表示
- ノードにホバーすると役割・スキル・中心性スコアを表示
- 右上の凡例に媒介中心性 TOP3 を表示

**媒介中心性について**  
業務フロー上で多くの経路を中継しているメンバーほどスコアが高くなります。  
スコアが 0.2 を超えるメンバーはボトルネック候補として赤枠で強調されます。  
そのメンバーが不在になった場合に業務フローが分断されるリスクがあります。

### スキルマップ（skill_map.html）

- ヒートマップ・表形式をタブで切り替えて表示
- チーム・スキルのドロップダウンフィルターに対応
- 保有者が2人以下のスキルをオレンジ色でリスク表示
- 上部サマリーにメンバー数・スキル種別数・平均スキル数・リスクスキル数を表示

## カスタマイズ

チームカラーは各スクリプト上部の `TEAM_COLORS` を編集してください。

```python
# team_graph.py / skill_map.py 共通
TEAM_COLORS = {
    "A": "#5B8CF0",  # チームA：青
    "B": "#E8773A",  # チームB：オレンジ
    "C": "#7B61E8",  # チームC：紫
}
```

ボトルネック判定のしきい値（デフォルト 0.2）は `team_graph.py` の以下の箇所で変更できます。

```python
border_color = "#E82C2C" if bc > 0.2 else "#ffffff"
border_width = 4         if bc > 0.2 else 1.5
```

スキル偏りリスクの判定人数（デフォルト 2人以下）は `skill_map.py` の以下の箇所で変更できます。

```python
const risk = cnt <= 2;
```
