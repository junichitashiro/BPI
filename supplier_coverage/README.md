# サプライヤ対応領域マップ

サプライヤごとの対応領域をヒートマップで可視化するツールです。  
受注窓口担当者が「どのサプライヤがどの加工領域に対応しているか」を一覧で確認し、発注先の検討や危険領域（対応サプライヤが少ない領域）の把握に使用します。

---

## ファイル構成

```
.
├── supplier_heatmap.html   # ヒートマップ表示ツール（ブラウザで開く）
├── supplier_coverage.xlsx  # データ管理ファイル（Excelで編集する）
└── README.md               # このファイル
```

---

## 使い方

1. `supplier_heatmap.html` と `supplier_coverage.xlsx` を同じフォルダに置く
2. `supplier_heatmap.html` をブラウザで開く
3. 「Excelを読み込む」ボタンをクリックし、`supplier_coverage.xlsx` を選択する
4. 画面上部のタブで表示する項目カテゴリを切り替えながら確認する

---

## Excelファイルの構成

### シート1：サプライヤマスタ

サプライヤの基本情報を管理します。

| 列名 | 内容 |
|------|------|
| サプライヤID | 一意のID（例：SUP001）。対応領域シートと紐付けるキーのため変更不可 |
| サプライヤ名 | 表示名 |
| 担当者 | 担当者名 |
| 電話 | 連絡先電話番号 |
| メール | 連絡先メールアドレス |
| 備考 | 自由記述 |

### シート2：対応領域

ヒートマップの元データです。1行1レコードで入力します。

| 列名 | 内容 |
|------|------|
| サプライヤID | サプライヤマスタのIDと一致させる |
| 項目カテゴリ | タブ名になる分類（例：素材、表面処理） |
| 項目名 | 具体的な項目（例：S45C、黒染め） |
| 対応値 | 下記ルールに従い入力 |

**対応値の入力ルール**

| 値 | 意味 | 対象カテゴリ |
|----|------|-------------|
| ○ | 対応可 | 素材・対応サイズ・表面処理・熱処理・ねじ |
| △ | 条件付き対応（要確認） | 同上 |
| × | 対応不可 | 全カテゴリ共通 |
| 数値（例：0.01） | 対応可能な最高精度（mm） | 幾何公差のみ |

---

## 幾何公差の表示ルール

幾何公差はExcelに「最高対応精度」を1値で入力します。  
ヒートマップ上では精度帯（0.005 / 0.01 / 0.02 / 0.05）ごとに列が展開され、入力値以下の精度帯すべてに自動で「○」が付きます。

**例：真円度 → `0.01` と入力した場合**

| 真円度（0.005） | 真円度（0.01） | 真円度（0.02） | 真円度（0.05） |
|:---:|:---:|:---:|:---:|
| × | ○ | ○ | ○ |

---

## 危険領域の判定

対応サプライヤ数が全体の40%以下の列を「危険領域」と判定します。  
該当列には「⚠」マークが付き、画面下部にアラートが表示されます。  
発注先の選択肢が限られるため、領域拡大の優先検討の目安として活用してください。

## データの追加・変更

### 項目の追加

「対応領域」シートに行を追加するだけで反映されます。  
「項目カテゴリ」列の名前がそのままタブ名になるため、表記を統一してください。

### サプライヤの追加

1. 「サプライヤマスタ」シートに新しいIDと名称を追加する
2. 「対応領域」シートに該当IDの行を追加する

### 精度帯の変更（幾何公差）

`supplier_heatmap.html` 内の以下の箇所を編集してください。

```javascript
const GEO_LEVELS = [0.005, 0.01, 0.02, 0.05];
```
